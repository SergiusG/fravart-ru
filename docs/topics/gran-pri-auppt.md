# Гран-При АУППТ

## Описание

Автоматическая установка порошкового пожаротушения (АУППТ).

Объект: БЦ «Гран-При», г. Воронеж, ул. Ворошилова, д. 1А.
Помещения: Кроссовая (пом. 28), Электрощитовая (пом. 33).
Проектировщик: ООО «Виан-Сеть», ГИП Вдовыченко В.В.
Подрядчик: ООО «НПЦ КСП».
