# ЭОМ-спецификации

## Описание

Извлечение спецификаций оборудования, изделий, материалов из проектной документации (PDF) для подготовки коммерческих предложений.

Применялось к проектам:
- Школа (Москва) — `502П-24-АНО-ИОС1` (515 стр, 70 МБ)
- Гран-При АУППТ — PDF проектировщика

## Артефакты

- Спецификация Школа Москва: https://docs.google.com/spreadsheets/d/1dSD8eG7qaM1iqSgYy9PMVFAevhHSigZJ/edit
- Спецификация Гран-При: https://docs.google.com/spreadsheets/d/15J5WSdq276Thd8vIolI6MQirs7gaV8Wq/edit

## Подход

1. Найти PDF на Google Drive (поиск по имени проекта или коду 502П-XX-...)
2. Скачать через `google_api.py drive download`
3. Извлечь текст через `pymupdf` (постранично)
4. Найти разделы «Спецификация оборудования» (обычно в конце тома)
5. Распарсить таблицу в pandas / openpyxl
6. Сохранить Excel с колонками: №, Наименование, Тип, Код, Производитель, Ед., Кол-во, Прим.

## Сравнение источников

Для Школы сравнили:
- `.doc` файлы от заказчика (3 шт: ЭОМ_ВРУ, ЭОМ_начало, ЭОМ_окончание) — 560 позиций, содержал дубликаты
- **PDF** первоисточник — 690 позиций, чище

**Вывод:** PDF первоисточник точнее. Использовали его.
