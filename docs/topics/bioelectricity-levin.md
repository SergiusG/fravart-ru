# Bioelectricity / Levin

## Описание

Научпоп-видео о биоэлектричестве по работам Michael Levin (Tufts University).

Концепция: «Тело думает без мозга». Биоэлектрическое поле — это программа, которая управляет формой и регенерацией тела. Мозг — лишь один из инструментов, а не главный контроллер.

Канал: «Архитектор Сознания» (YouTube).
