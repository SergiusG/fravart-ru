# Что делали

Распарсили `.doc` спецификации от заказчика в Excel. Три файла → два Excel:

- `Спецификация_ЭОМ_ВРУ.xlsx` — 21 позиция (щитовое оборудование ВРУ)
- `Спецификация_ЭОМ.xlsx` — 560 позиций (щиты, кабели, трубы, лотки, светильники)

## Технические детали

- Файлы `.doc` содержат таблицы с разделителем `\x07` (BEL), не `\t`
- Написали парсер `parse_to_xlsx.py` с правильной обработкой заголовков секций
- Логика склейки «наименование» + «хвостовая часть с кодом/manuf/unit/qty»

## Результаты

- ✅ Получили структурированный Excel
- ⚠️ Подозрение: данные из .doc неточные (позже выяснится при сравнении с PDF)
- Добавили столбцы «Стоимость работ» с формулами
