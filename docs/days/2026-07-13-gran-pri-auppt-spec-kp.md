# Что делали

Извлекли спецификацию из PDF проекта АУППТ (порошковое пожаротушение) для БЦ Гран-При в Воронеже.

## Объект

- БЦ «Гран-При», г. Воронеж, ул. Ворошилова, д. 1А
- Помещения: Кроссовая (пом. 28) + Электрощитовая (пом. 33)
- Проектировщик: ООО «Виан-Сеть», ГИП Вдовыченко В.В.

## Что конкретно

1. Нашли 2 PDF (Кроссовая + Электрощитовая) на Drive
2. Скачали, распарсили спецификацию (стр. 38 — таблица расчёта АКБ)
3. Собрали 27 позиций (14 Кроссовая + 13 Электрощитовая)
4. Поставили цены оборудования и монтажа
5. Создали КП (4 листа)

## Правки по ходу

- Добавили столбцы «Цена оборудования, руб/шт» и «Стоимость оборудования, руб» (было только монтаж)
- Восстановили форматирование (жирные заголовки, синий фон, границы, формулы)
