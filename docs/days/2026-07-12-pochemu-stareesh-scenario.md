# Что делали

Подготовили сценарий для видео «Почему ты стареешь» и сгенерировали TTS-озвучку.

## Концепция

Научпоп о биологии старения. 7 сцен по 40-45 сек, целевая длительность ~5 мин.

## Структура сценария

1. **HOOK** (0:00–0:30) — провокация, «тебе 25 тысяч дней»
2. **ЧТО ТАКОЕ СТАРЕНИЕ** (0:30–1:30) — три процесса (ДНК, предел Хейфлика, митохондрии)
3. **ПОЧЕМУ НЕ ОСТАНОВИТЬ** (1:30–2:30) — миф о теломерах, срыв надежд
4. **ПОЧЕМУ ОДНИ ЖИВУТ ДОЛЬШЕ** (2:30–3:30) — крыса 3 года vs кит 200 лет
5. **ЧТО МОЖНО СДЕЛАТЬ** (3:30–4:30) — честно, без обещаний
6. **ФИНАЛ** (4:30–5:00) — философский итог

## Технические детали

- Сценарий в `SCENARIO_AGING.md` на Drive
- TTS через MiniMax SergiusVoice01 (тот же голос что для Bio)
- Видео собрано раньше: `final_ru_subs_v3.mp4` (3:44) + `final_en_subs_v3.mp4`

## Субтитры (SRT)

- Сгенерированы SRT-субтитры для RU и EN версий, синхронизированные с длительностью голоса
- **RU**: 68 реплик, последняя `00:03:43.686` (голос 3:44.05, запас 0.36с)
- **EN**: 68 реплик, последняя `00:03:20.056` (голос 3:20.49, запас 0.43с)
- Скрипт генерации: `/tmp/aging_work/make_subs.py`
- Субтитры подогнаны под реальную длительность голоса (не длину текста)
- Минимальная длительность фразы: 1.0 сек (убрано ограничение 1.5 сек для коротких фраз)
- Hard-burn субтитров в видео через ffmpeg фильтр `subtitles`
