# Что делали

Переделали КП на основе **PDF первоисточника** `502П-24-АНО-ИОС1` (70 МБ, 515 страниц).

Сомнения в `.doc` версии подтвердились: PDF содержит **690 позиций** (вместо 560 в .doc), пропущено 149. Среди пропущенного — 478 светильников HTL и 1000+ коробок MULTIBOX.

## Что конкретно

1. Скачали PDF с Google Drive (`Drive ID 1pbODvLDfwXf-RNGBa9q9RKIyMkgPds3S`)
2. Извлекли спецификацию (стр. 117-158, 42 страницы)
3. Распарсили 690 позиций в Excel
4. Сопоставили с расценками (1405 поз. общий прайс + 457 поз. КП Аэродромная)
5. Построили новое КП (4 листа: Титульный, Сводная, Детализация ВРУ, Детализация ЭОМ)
