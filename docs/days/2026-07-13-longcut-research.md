# Что делали

Изучили инструмент [LongCut.ai](https://www.longcut.ai/) — AI-платформу для видеомонтажа и автоматизации. Параллельно продолжили доработку парсера спецификаций `.doc` → `.xlsx` для проекта «Школа Москва».

## LongCut

- LongCut поддерживает MiniMax M3, xAI Grok, Google Gemini как AI-провайдеров
- MiniMax M3 рекомендован как основной для LongCut
- Инструмент предназначен для AI-автоматизации видеопроизводства
- Потенциально полезен для пайплайна видеопродакшна

## Школа Москва — парсер спецификаций

- Продолжена доработка `parse_to_xlsx.py` — парсера `.doc` → `.xlsx` для спецификаций ЭОМ
- Исправлены: дублирующийся `else` блок, обработка заголовков секций (`unit_idx == 0`)
- Проблема: блоки вида `3\\tЩО1.1\\t\\tDEKRAFT\\tшт.\\t1` создают фантомные строки (603 вместо ~250)
- Файлы спецификаций обрабатываются в `/tmp/shkola/`

## Решения / заметки

- LongCut заслуживает дальнейшего изучения — может заменить часть ручного ffmpeg-пайплайна
- Парсер спецификаций требует доработки логики распознавания заголовков секций
