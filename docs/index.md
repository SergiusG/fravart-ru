Личная вики-страница: архив наших с AI-агентом совместных сессий.

## О чём этот сайт

Здесь собраны записи о работе, которую я делаю с AI-агентом (Hermes):

- Проекты НПЦ КСП (строительство, КП, сметы)
- Видеопродакшн (канал «Архитектор Сознания»)
- Настройка инструментов и конфигурации

Каждый день работы = одна запись в [Записях по дням](days.md). Каждая тема = страница с описанием и хронологией.
